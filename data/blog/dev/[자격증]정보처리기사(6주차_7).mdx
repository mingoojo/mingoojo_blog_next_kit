### 1. 데이터 타입 (Data Type)

1. **데이터 타입이란?**

   > **한 줄 정의:** 변수에 저장될 데이터의 **형식을 나타내는 것** (문자형, 정수형, 실수형 등)

2. **데이터 타입 유형 6가지**

   | 유형                 | 한 줄 정의                 | 예시               |
   | -------------------- | -------------------------- | ------------------ |
   | **정수 타입**        | **소수점 없는** 숫자       | 1, -1, 10, -100    |
   | **부동 소수점 타입** | **소수점 있는** 실수       | 0.123×10², -1.6×2³ |
   | **문자 타입**        | **한 문자** 저장 (**' '**) | 'A', 'a', '1'      |
   | **문자열 타입**      | **문자열** 저장 (**" "**)  | "Hello!", "1+2=3"  |
   | **불린 타입**        | **참/거짓** 저장           | true, false        |
   | **배열 타입**        | **같은 타입 집합** 저장    | \{1, 2, 3, 4, 5}   |
   - **문자 vs 문자열**
     - **문자** = 작은따옴표 **' '** = 한 글자
     - **문자열** = 큰따옴표 **" "** = 여러 글자

   - **불린 타입**
     - 기본값 = **false (거짓)**

3. **C/C++ 데이터 타입**
   - 문자형

     | 타입          |   크기    | 범위       |
     | ------------- | :-------: | ---------- |
     | **char**      | **1Byte** | -128 ~ 127 |
     | unsigned char |   1Byte   | 0 ~ 255    |

   - 정수형

     | 타입      |   크기    | 범위              |
     | --------- | :-------: | ----------------- |
     | **short** | **2Byte** | -32,768 ~ 32,767  |
     | **int**   | **4Byte** | 약 -21억 ~ 21억   |
     | **long**  | **4Byte** | 약 -21억 ~ 21억   |
     | long long |   8Byte   | 약 -922경 ~ 922경 |

   - 실수형

     | 타입        |   크기    |
     | ----------- | :-------: |
     | **float**   | **4Byte** |
     | **double**  | **8Byte** |
     | long double |   8Byte   |

   - 논리형

     | 타입     |   크기    | 값           |
     | -------- | :-------: | ------------ |
     | **bool** | **1Byte** | true / false |

4. **Java 데이터 타입**
   - 문자형

     | 타입     |   크기    | 특징          |
     | -------- | :-------: | ------------- |
     | **char** | **2Byte** | 유니코드 지원 |

   - 정수형

     | 타입      |   크기    |
     | --------- | :-------: |
     | **byte**  | **1Byte** |
     | **short** | **2Byte** |
     | **int**   | **4Byte** |
     | **long**  | **8Byte** |

   - 실수형

     | 타입       |   크기    |
     | ---------- | :-------: |
     | **float**  | **4Byte** |
     | **double** | **8Byte** |

   - 논리형

     | 타입        |   크기    |
     | ----------- | :-------: |
     | **boolean** | **1Byte** |

5. **C/C++ vs Java 비교**

   |   구분   |   C/C++   |    Java     |
   | :------: | :-------: | :---------: |
   | **char** | **1Byte** |  **2Byte**  |
   |  short   |   2Byte   |    2Byte    |
   |   int    |   4Byte   |    4Byte    |
   |   long   |   4Byte   |  **8Byte**  |
   |  논리형  | **bool**  | **boolean** |

   💡 **차이점 암기:**
   - **char**: C는 1Byte, **Java는 2Byte** (유니코드)
   - **long**: C는 4Byte, **Java는 8Byte**
   - **논리형**: C는 **bool**, Java는 **boolean**

6. **Python 데이터 타입**

   | 타입        |    크기    | 특징   |
   | ----------- | :--------: | ------ |
   | **str**     | **무제한** | 문자열 |
   | **int**     | **무제한** | 정수   |
   | **float**   |   8Byte    | 실수   |
   | **complex** |   16Byte   | 복소수 |

   💡 **Python 특징:** str, int는 크기 **무제한!**

7. **Python 시퀀스 자료형**

   |      자료형       | 한 줄 정의                                  | 변경 |
   | :---------------: | ------------------------------------------- | :--: |
   | **리스트 (List)** | 다양한 자료형 연속 저장, **크기 변경 가능** |  ⭕  |
   | **튜플 (Tuple)**  | 요소 연속 저장, **변경 불가**               |  ❌  |
   |     **range**     | **연속된 숫자** 생성                        |  -   |

8. **리스트 vs 튜플**

   |      구분      |  리스트  |    튜플    |
   | :------------: | :------: | :--------: |
   |      기호      | **[ ]**  |  **( )**   |
   | 추가/삭제/변경 | **가능** | **불가능** |

### 2. 변수 (Variable)

1. **변수란?**

   > **한 줄 정의:** 명령 처리 중 발생하는 값을 저장하기 위한 **공간** (변할 수 있는 값)

2. **변수명 작성 규칙**

   | 규칙               | 설명                            | 예시               |
   | ------------------ | ------------------------------- | ------------------ |
   | **사용 가능 문자** | 영문자, 숫자, **\_(언더바)**    | abc, \_abc, abc123 |
   | **첫 글자**        | 영문자 또는 **\_** (숫자 불가!) | ❌ 2abc            |
   | **글자 수**        | 제한 없음                       | -                  |
   | **특수문자**       | **공백, \*, +, -, / 불가**      | ❌ sum\*, ha p     |
   | **대소문자**       | **구분함**                      | Kim ≠ kim          |
   | **예약어**         | **사용 불가**                   | ❌ for, int, if    |
   | **문장 끝**        | **세미콜론(;)** 필수            | int a = 5;         |
   - 변수명 예제

     |  변수명  | 사용 가능? | 이유                           |
     | :------: | :--------: | ------------------------------ |
     |   2abc   |     ❌     | **숫자로 시작**                |
     |  sum\*   |     ❌     | **특수문자** 사용              |
     |   for    |     ❌     | **예약어**                     |
     |   ha p   |     ❌     | **공백** 사용                  |
     | Kim, kim |     ⭕     | 대소문자 구분 → 서로 다른 변수 |

3. **헝가리안 표기법 (Hungarian Notation)**

   > **한 줄 정의:** 변수명에 **데이터 타입을 명시**하는 표기법

   ```c
   int iCount;    // i = int
   float fValue;  // f = float
   char cGrade;   // c = char
   ```

4. **예약어 (Reserved Word)**

   > **한 줄 정의:** 이미 용도가 정해져 있어 **변수명으로 사용 불가**한 단어
   - C언어 예약어

     | 구분            | 예약어                                                                                      |
     | --------------- | ------------------------------------------------------------------------------------------- |
     | **반복문**      | do, for, while                                                                              |
     | **선택문**      | case, default, else, if, switch                                                             |
     | **분기문**      | break, continue, goto, return                                                               |
     | **자료형**      | char, double, enum, float, int, long, short, signed, struct, typedef, union, unsigned, void |
     | **기억 클래스** | auto, extern, register, static                                                              |
     | **기타**        | const, sizeof, volatile                                                                     |

5. **기억 클래스 (Storage Class)**

   > **한 줄 정의:** 변수의 **기억영역을 결정**하는 작업
   - 5가지 기억 클래스

     | 종류                |    예약어    |    기억영역    |  생존기간  |  사용범위  |
     | ------------------- | :----------: | :------------: | :--------: | :--------: |
     | **자동 변수**       |   **auto**   |  메모리(스택)  | **일시적** | **지역적** |
     | **레지스터 변수**   | **register** |  **레지스터**  | **일시적** | **지역적** |
     | **정적 변수(내부)** |  **static**  | 메모리(데이터) | **영구적** | **지역적** |
     | **정적 변수(외부)** |  **static**  | 메모리(데이터) | **영구적** | **전역적** |
     | **외부 변수**       |  **extern**  | 메모리(데이터) | **영구적** | **전역적** |

   - 기억 클래스 상세
     1. 자동 변수 (auto)
        - 함수/블록 내에서 선언, 벗어나면 **자동 소멸**
        - 초기화 안 하면 → **쓰레기값(Garbage Value)**
     2. 레지스터 변수 (register)
        - **CPU 레지스터**에 저장 → **처리 속도 향상**
        - 레지스터 부족 시 → **자동 변수로 취급**
        - **주소 연산자(&) 사용 불가!**
     3. 정적 변수 (static)
        - 함수/블록 종료 후에도 **값 유지**
        - 초기화 안 하면 → **자동으로 0**
        - 초기화는 **한 번만** 가능
     4. 외부 변수 (extern)
        - **다른 파일**의 변수/함수 참조
        - 함수 **밖**에서 선언
        - 초기화 안 하면 → **자동으로 0**
        - 다른 파일 변수 참조 시 **초기화 불가**
   - 초기화 비교

     | 변수 종류         | 초기화 안 하면? |
     | ----------------- | :-------------: |
     | **자동 변수**     |  **쓰레기값**   |
     | **레지스터 변수** |  **쓰레기값**   |
     | **정적 변수**     |      **0**      |
     | **외부 변수**     |      **0**      |

6. **const와 리터럴**
   - const (상수화)
     - 변수를 **상수로 만드는** 예약어

     ```c
     const float PI = 3.1415927;
     // PI는 이후 변경 불가!
     ```

   - 리터럴 (Literal)
     - **값 그 자체** (3.14, "Hello" 등)
   - 심볼릭 상수 (Symbolic Constant)
     - **이름을 붙인 상수** (const PI)

7. **가비지 콜렉터 (Garbage Collector)**
   - 사용하지 않는 변수의 메모리를 **강제 해제**하는 모듈
     - **가비지 콜렉션**: 메모리 해제 작업
     - **가비지 콜렉터**: 작업 수행 모듈

8. **변수 선언 형식**

   ```c
   자료형 변수명 = 값;
   ```

   ```c
   int a = 5;      // 정수형 변수 a에 5 저장
   char aa = 'A';  // 문자형 변수 aa에 'A' 저장
   float fl = 24.56f;  // 실수형 변수 fl에 24.56 저장
   ```

### 3. 연산자 (Operator)

1. **연산자란?**

   > **한 줄 정의:** 데이터를 처리하여 결과를 얻기 위해 사용하는 **기호**

2. **연산자 종류 7가지**

   | 연산자          | 한 줄 정의                 |
   | --------------- | -------------------------- |
   | **산술 연산자** | 산술 계산 (+, -, \*, /, %) |
   | **관계 연산자** | 두 수 비교 → 참/거짓       |
   | **비트 연산자** | 비트별(0, 1) 연산          |
   | **논리 연산자** | 논리 값 연산 → 참/거짓     |
   | **대입 연산자** | 연산 후 결과 대입          |
   | **조건 연산자** | 조건에 따라 다른 수식 수행 |
   | **기타 연산자** | sizeof, 콤마, cast         |
   1. 산술 연산자
      | 연산자 | 의미 |
      | :----: | --------------- |
      | **+** | 덧셈 |
      | **-** | 뺄셈 |
      | **\*** | 곱셈 |
      | **/** | 나눗셈 |
      | **%** | **나머지** |
      | **++** | **증가** 연산자 |
      | **--** | **감소** 연산자 |
   - 증감 연산자 위치

     |  형식   | 동작                    |
     | :-----: | ----------------------- |
     | **++a** | **먼저** 증가 후 사용   |
     | **a++** | 사용 후 **나중에** 증가 |
   2. 관계 연산자
      - 두 수를 비교하여 **참(1) 또는 거짓(0)** 반환

      | 연산자  | 의미        |
      | :-----: | ----------- |
      | **==**  | 같다        |
      | **!=**  | 같지 않다   |
      |  **>**  | 크다        |
      | **>=**  | 크거나 같다 |
      | **\<**  | 작다        |
      | **\<=** | 작거나 같다 |

      💡 **참고:** 거짓 = 0, 참 = 1 (0 외의 모든 숫자도 참)

   3. 비트 연산자

      |  연산자  |     의미      | 설명                   |
      | :------: | :-----------: | ---------------------- |
      |  **&**   |    **AND**    | 모두 1일 때만 1        |
      |  **\|**  |    **OR**     | 하나라도 1이면 1       |
      |  **^**   |    **XOR**    | 같으면 0, 다르면 1     |
      |  **~**   |    **NOT**    | 0↔1 반전              |
      | **\<\<** |  왼쪽 시프트  | 비트를 왼쪽으로 이동   |
      |  **>>**  | 오른쪽 시프트 | 비트를 오른쪽으로 이동 |
      - 비트 연산 예시

      ```
        1010 (10)
      & 1100 (12)
      ---------
        1000 (8)   ← AND: 둘 다 1일 때만 1

        1010 (10)
      | 1100 (12)
      ---------
        1110 (14)  ← OR: 하나라도 1이면 1

        1010 (10)
      ^ 1100 (12)
      ---------
        0110 (6)   ← XOR: 다르면 1
      ```

   4. 논리 연산자

      |  연산자  |  의미   | 설명               |
      | :------: | :-----: | ------------------ |
      |  **!**   | **NOT** | 부정               |
      |  **&&**  | **AND** | 모두 참이면 참     |
      | **\|\|** | **OR**  | 하나라도 참이면 참 |
   - 비트 vs 논리 연산자 비교

     | 구분 | 비트 연산자 | 논리 연산자 |
     | :--: | :---------: | :---------: |
     | AND  |    **&**    |   **&&**    |
     |  OR  |   **\|**    |  **\|\|**   |
     | NOT  |    **~**    |    **!**    |
   5. 대입 연산자
      - 연산 후 결과를 **대입**하는 연산자

      |  연산자   | 예시      | 의미         |
      | :-------: | --------- | ------------ |
      |  **+=**   | a += 1    | a = a + 1    |
      |  **-=**   | a -= 1    | a = a - 1    |
      |  **\*=**  | a \*= 1   | a = a \* 1   |
      |  **/=**   | a /= 1    | a = a / 1    |
      |  **%=**   | a %= 1    | a = a % 1    |
      | **\<\<=** | a \<\<= 1 | a = a \<\< 1 |
      |  **>>=**  | a >>= 1   | a = a >> 1   |

   6. 조건 연산자 (삼항 연산자)
      - 조건에 따라 **다른 수식** 수행
      - 형식

        ```c
        조건 ? 수식1 : 수식2
        ```

        - 조건이 **참** → **수식1** 실행
        - 조건이 **거짓** → **수식2** 실행

      - 예시

        ```c
        int max = (a > b) ? a : b;
        // a가 크면 a, 아니면 b
        ```

   7. 기타 연산자

      |    연산자    | 의미 | 설명                           |
      | :----------: | ---- | ------------------------------ |
      |  **sizeof**  | 크기 | 자료형의 **크기(바이트)** 표시 |
      |    **,**     | 콤마 | 순서대로 수행 (순서 연산자)    |
      | **(자료형)** | cast | **형 변환** 연산자             |

3. **cast 연산자 예시**

   ```c
   a = (int)1.3 + (int)1.4;
   // 1 + 1 = 2 (소수점 버림)
   ```

4. **연산자 우선순위**

   | 우선순위 | 대분류     | 연산자           | 결합규칙 |
   | :------: | ---------- | ---------------- | :------: |
   |    1     | **단항**   | ! ~ ++ -- sizeof |    ←     |
   |    2     | **산술**   | \* / %           |    →     |
   |    3     | **산술**   | + -              |    →     |
   |    4     | **시프트** | \<\< >>          |    →     |
   |    5     | **관계**   | < \<= >= >       |    →     |
   |    6     | **관계**   | == !=            |    →     |
   |    7     | **비트**   | & ^ \|           |    →     |
   |    8     | **논리**   | && \|\|          |    →     |
   |    9     | **조건**   | ? :              |    →     |
   |    10    | **대입**   | = += -= 등       |    ←     |
   |    11    | **순서**   | ,                |    →     |

### 4. 데이터 입출력

1. **표준 입출력 함수란?**

   > **한 줄 정의:** **키보드로 입력**받아 **화면으로 출력**할 때 사용하는 함수

2. **C언어 표준 입출력 함수**

   |   구분   | 함수          | 한 줄 정의               |
   | :------: | ------------- | ------------------------ |
   | **입력** | **scanf()**   | 서식에 맞게 **입력**     |
   | **입력** | **getchar()** | **한 문자** 입력         |
   | **입력** | **gets()**    | **문자열** 입력          |
   | **출력** | **printf()**  | 서식에 맞게 **출력**     |
   | **출력** | **putchar()** | **한 문자** 출력         |
   | **출력** | **puts()**    | **문자열** 출력 + 줄바꿈 |
   1. scanf() 함수 : 키보드로 입력받아 **변수에 저장**하는 함수
      - 문법
        - **%** : 서식 문자 지정
        - **3** : 입력 자릿수 3자리
        - **d** : 10진수로 입력
        - **&a** : 변수 a의 **주소**에 저장
      - 형식

        ```c
        scanf(서식 문자열, 변수의 주소)
        ```

      - 예시

        ```c
        scanf("%3d", &a);
        ```

      - 특징
        - 입력 자료형, 자릿수 지정 가능
        - 여러 개 데이터 동시 입력 가능
        - **주소 연산자 & 필수!**

   2. printf() 함수
      - 인수로 주어진 값을 **화면에 출력**하는 함수
      - 형식
        ```c
        printf(서식 문자열, 변수)
        ```
      - 예시

        ```c
        printf("%-8.2f", 200.2);
        // 결과: 200.20▢▢ (왼쪽 정렬, 8자리, 소수점 2자리)
        ```

      - 문법
        - **%** : 서식 문자 지정
        - **-** : **왼쪽** 정렬
        - **8** : 출력 자릿수 8자리
        - **.2** : 소수점 이하 2자리
        - **f** : 실수로 출력

   3. scanf vs printf

   |    구분     |   scanf    |    printf    |
   | :---------: | :--------: | :----------: |
   |    용도     |    입력    |     출력     |
   | 주소 연산자 | **& 필수** | **& 불필요** |

3. **서식 문자열**

   |  서식   | 의미              | 설명         |
   | :-----: | ----------------- | ------------ |
   | **%d**  | **정수** 10진수   | decimal      |
   | **%u**  | **부호없는** 정수 | unsigned     |
   | **%o**  | **8진수**         | octal        |

   | **%ld** | **long형** 10진수 | long decimal |
   | **%p**  | **주소** (16진수) | pointer      |

4. **제어문자 (Escape Sequence)**

   | 제어문자 | 의미            | 설명             |

   |  **\a**  | **벨 소리**     | alert            |
   |  **\f**  | **페이지 넘김** | form feed        |

5. **기타 입출력 함수**
   1. 입력 함수

      | 함수          | 한 줄 정의                  |
      | ------------- | --------------------------- |
      | **getchar()** | **한 문자** 입력            |
      | **gets()**    | **문자열** 입력 (Enter까지) |

   2. 출력 함수

### 5. 제어문

1. **제어문이란?**

   > **한 줄 정의:** 프로그램의 **실행 순서를 변경**할 때 사용하는 명령문

2. **제어문 종류**

   | 종류          | 한 줄 정의                      |
   | ------------- | ------------------------------- |
   | **단순 if문** | 조건 **한 개**일 때 분기        |
   | **다중 if문** | 조건 **여러 개**일 때 분기      |
   | **switch문**  | 분기 **여러 곳**일 때 간단 처리 |
   | **goto문**    | 원하는 문장으로 **건너뜀**      |
   | **반복문**    | 조건 만족 시 **반복** 실행      |
   1. 단순 if문
      - 조건이 **한 개**일 때 사용하는 제어문
      - 형식 1: 조건이 참일 때만 실행

        ```c
        // 실행 문장 1개
        if(조건)
            실행할 문장;

        // 실행 문장 2개 이상 → 중괄호 { } 필수
        if(조건)
        {
            실행할 문장1;
            실행할 문장2;
        }
        ```

      - 형식 2: 참/거짓 각각 다른 문장 실행

        ```c
        if(조건)
            실행할 문장1;  // 참일 때
        else
            실행할 문장2;  // 거짓일 때
        ```

   2. 다중 if문
      - 조건이 **여러 개**일 때 사용하는 제어문
      - 형식 1: else if 사용

        ```c
        if(조건1)
            실행할 문장1;
        else if(조건2)
            실행할 문장2;
        else if(조건3)
            실행할 문장3;
        else
            실행할 문장4;  // 모든 조건 거짓일 때
        ```

      - 형식 2: 중첩 if문 (if문 안에 if문)

        ```c
        if(조건1)
        {
            if(조건2)
                실행할 문장1;
            else
                실행할 문장2;
        }
        else
            실행할 문장3;
        ```

   3. switch문
      - 분기할 곳이 **여러 곳**일 때 간단하게 처리하는 제어문
      - 형식

        ```c
        switch(수식)
        {
            case 레이블1:
                실행할 문장1;
                break;
            case 레이블2:
                실행할 문장2;
                break;
            default:
                실행할 문장3;  // 해당 case 없을 때
        }
        ```

      - switch문 특징

        | 요소             | 역할                       |
        | ---------------- | -------------------------- |
        | **switch(수식)** | 비교할 값                  |
        | **case 레이블:** | 수식과 비교할 값           |
        | **break;**       | switch문 **탈출** (필수!)  |
        | **default:**     | 해당 case **없을 때** 실행 |

      - break 없으면?
        - 다음 case까지 **연속 실행** (Fall-through)

   4. goto문
      - 원하는 다른 문장으로 **건너뛰어** 수행하는 제어문
      - 형식

        ```c
        goto 레이블;
            ...
        레이블:
            실행할 문장;
        ```

      - goto문 특징
        - 원하는 곳으로 **쉽게 이동** 가능
        - 많이 사용 시 **유지보수 어려움** → **거의 사용 안 함!**

   5. if문 vs switch문 비교

      |   구분    |          if문          |       switch문        |
      | :-------: | :--------------------: | :-------------------: |
      |   조건    | **범위, 복잡한 조건**  |   **특정 값** 비교    |
      | 비교 연산 | >, \<, >=, \<= 등 가능 |    **== 만** 가능     |
      |  가독성   |    조건 많으면 복잡    | 조건 많을 때 **간단** |

3. **제어문 흐름도**
   1. 단순 if문

      ```
          [조건]
          /     \
        참       거짓
        ↓         ↓
      [문장1]   [문장2]
      ```

   2. switch문

      ```
          [수식]
        /  |  \
      case1 case2 default
        ↓    ↓     ↓
      문장1 문장2  문장3
      ```

### 6. 반복문

1. **반복문이란?**

   > **한 줄 정의:** 일정한 횟수를 **반복**하는 명령문 (조건이 참인 동안 반복)

2. **반복문 종류 3가지**

   | 반복문       | 한 줄 정의                       |
   | ------------ | -------------------------------- |
   | **for**      | **횟수** 정해진 반복             |
   | **while**    | **조건** 참인 동안 반복          |
   | **do~while** | **무조건 1번** 실행 후 조건 판단 |
   1. for문
      - 초기값, 최종값, 증가값을 지정하여 **정해진 횟수** 반복
      - 형식

        ```c
        for(초기값; 조건식; 증가값)
            실행할 문장;
        ```

      - 예시

        ```c
        for(int i = 1; i <= 5; i++)
            printf("%d ", i);
        // 출력: 1 2 3 4 5
        ```

      - for문 특징

        | 특징          | 설명                                     |
        | ------------- | ---------------------------------------- |
        | **생략 가능** | 초기값, 조건식, 증가값 생략 가능         |
        | **복수 지정** | 각 요소에 여러 수식 가능                 |
        | **문자 사용** | 'A'~'Z' 같은 문자도 가능                 |
        | **조건 거짓** | 처음부터 거짓이면 **한 번도 실행 안 함** |

      - for문 다양한 예시

        ```c
        // 증가값 생략
        for(a = 1; sum <= 30;) sum += a;

        // 복수 지정
        for(a = 0; sum <= 10; a++, sum += a)

        // 모두 생략 → 무한 반복!
        for(;;)

        // 문자 사용
        for(char a = 'A'; a <= 'Z'; a++)
        ```

      - for(;;) = **무한 반복!**

   2. while문
      - 조건이 **참인 동안** 반복 수행
      - 형식

        ```c
        while(조건)
            실행할 문장;
        ```

      - 예시

        ```c
        int i = 1;
        while(i <= 5)
        {
            printf("%d ", i);
            i++;
        }
        // 출력: 1 2 3 4 5
        ```

      - while문 특징
        - 조건이 **처음부터 거짓**이면 **한 번도 실행 안 함**
        - 조건이 **거짓**이 되면 반복 종료

   3. do~while문
      - **무조건 1번 실행** 후 조건 판단하여 반복
      - 형식

        ```c
        do
            실행할 문장;
        while(조건);  // 세미콜론(;) 필수!
        ```

      - 예시

        ```c
        int i = 1;
        do
        {
            printf("%d ", i);
            i++;
        } while(i <= 5);
        // 출력: 1 2 3 4 5
        ```

      - do~while문 특징
        - **무조건 1번**은 실행 (조건 판단 전에 실행)
        - while 뒤에 **세미콜론(;) 필수!**

3. **while vs do~while 비교**

   |   구분    |         while          |       do~while        |
   | :-------: | :--------------------: | :-------------------: |
   | 조건 판단 |     **먼저** 판단      |    **나중에** 판단    |
   | 최소 실행 | **0번** (조건 거짓 시) |   **1번** (무조건)    |
   | 세미콜론  |          없음          | **while(조건);** 필수 |
   - 조건이 처음부터 거짓일 때

     ```c
     // while: 한 번도 실행 안 함
     int i = 10;
     while(i < 5)
         printf("실행");  // 실행 안 됨

     // do~while: 1번은 실행
     int i = 10;
     do
         printf("실행");  // 1번 실행됨
     while(i < 5);
     ```

4. **break와 continue**

   |    예약어    | 한 줄 정의                     | 사용 위치      |
   | :----------: | ------------------------------ | -------------- |
   |  **break**   | 블록을 **벗어남** (탈출)       | switch, 반복문 |
   | **continue** | 처음으로 **되돌아감** (건너뜀) | **반복문만**   |
   - break 예시
     ```c
     for(int i = 1; i <= 10; i++)
     {
         if(i == 5) break;  // i가 5일 때 탈출
         printf("%d ", i);
     }
     // 출력: 1 2 3 4
     ```
   - continue 예시

     ```c
     for(int i = 1; i <= 5; i++)
     {
         if(i == 3) continue;  // i가 3일 때 건너뜀
         printf("%d ", i);
     }
     // 출력: 1 2 4 5
     ```

5. **반복문 비교 총정리**

   |   구분    |       for       |     while     |     do~while      |
   | :-------: | :-------------: | :-----------: | :---------------: |
   |   사용    | **횟수** 정해짐 | **조건** 기반 | **최소 1번** 필요 |
   | 조건 판단 |      먼저       |     먼저      |     **나중**      |
   | 최소 실행 |       0번       |      0번      |      **1번**      |

### 7. 배열과 문자열

1. **배열이란?**

   > **한 줄 정의:** 동일한 데이터 유형을 여러 개 **하나의 이름**으로 조합해서 사용하는 것

2. **배열 핵심 개념**

   | 항목      | 내용                  |
   | --------- | --------------------- |
   | 위치 지정 | **첨자(인덱스)** 사용 |
   | 시작 위치 | **0**부터 시작        |
   | 기호      | **대괄호 [ ]**        |
   | 할당 순서 | **행 우선**           |

3. **1차원 배열**
   - 변수들을 **일직선상**으로 조합한 배열
   - 형식

     ```c
     자료형 변수명[개수];
     ```

   - 예시

     ```c
     int a[5];           // 5개 요소를 갖는 정수형 배열
     int a[5] = {1, 2, 3, 4, 5};  // 초기화
     ```

   - 메모리 구조

     ```
     a[0]  a[1]  a[2]  a[3]  a[4]
     1     2     3     4     5
     ```

   💡 **주의:** 첨자는 **0**부터 시작! (a[0] ~ a[4])

4. **2차원 배열**
   - 변수들을 **행과 열(평면)**로 조합한 배열
   - 형식

     ```c
     자료형 변수명[행개수][열개수];
     ```

   - 예시

     ```c
     int a[2][3];  // 2행 3열 배열
     int a[2][3] = {{1, 2, 3}, {4, 5, 6}};  // 초기화
     ```

   - 메모리 구조 (행 우선 할당)

     ```
             열0  열1  열2
     행0 →   1    2    3
     행1 →   4    5    6

     메모리: 1 → 2 → 3 → 4 → 5 → 6 (행 우선)
     ```

5. **배열 초기화**
   - 규칙

     | 상황            | 결과                      |
     | --------------- | ------------------------- |
     | **크기 생략**   | 초기값 개수만큼 배열 생성 |
     | **초기값 부족** | 나머지는 **0**으로 채움   |

   - 예시

     ```c
     int a[] = {1, 2, 3};        // 크기 3인 배열 생성
     int a[5] = {1, 2};          // a = {1, 2, 0, 0, 0}
     int a[5] = {0};             // 모든 요소 0으로 초기화
     ```

6. **배열 형태의 문자열**
   - C언어에서 문자열을 **배열**로 저장하는 방법
   - 형식

     ```c
     char 배열이름[크기] = "문자열";
     ```

   - 예시

     ```c
     char str[6] = "Hello";
     ```

   - 메모리 구조

     ```
     str[0] str[1] str[2] str[3] str[4] str[5]
       'H'   'e'    'l'    'l'    'o'   '\0'
     ```

7. **문자열 특징**

   | 특징          | 설명                             |
   | ------------- | -------------------------------- |
   | **널 문자**   | 문자열 끝에 **'\0'** 자동 삽입   |
   | **크기 지정** | 널 문자 포함해서 +1 해야 함      |
   | **저장 시점** | **선언 시에만** 문자열 저장 가능 |
   - 주의사항

     ```c
     char str[6] = "Hello";  // ⭕ 선언 시 초기화
     str = "World";          // ❌ 이미 선언된 배열에 저장 불가!
     ```

8. **Java 배열**
   - 배열 선언

     ```java
     int a[] = new int[5];  // 크기 5인 정수형 배열
     ```

   - 예시

     ```java
     public class Example {
         public static void main(String[] args) {
             int a[] = new int[5];
             int i;

             for (i = 0; i < 5; i++)
                 a[i] = i + 10;

             for (i = 0; i < 5; i++)
                 System.out.printf("%d ", a[i]);
         }
     }
     // 출력: 10 11 12 13 14
     ```

   - Java 배열 특징
     - 배열을 **객체**로 취급
     - **향상된 for문** 사용 가능
   - C언어 vs Java 배열 비교

     |    구분     |    C언어    |         Java          |
     | :---------: | :---------: | :-------------------: |
     |    선언     |  int a[5];  | int a[] = new int[5]; |
     | 시작 인덱스 |    **0**    |         **0**         |
     |    취급     | 메모리 공간 |       **객체**        |
