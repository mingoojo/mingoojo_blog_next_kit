### 8. 포인터

1. **포인터란?**

   > **한 줄 정의:** **변수의 주소**를 저장하는 것 (주소를 제어하는 기능)

2. **포인터 핵심 개념**

   | 용어            | 한 줄 정의               |
   | --------------- | ------------------------ |
   | **포인터**      | 변수의 **주소**          |
   | **포인터 변수** | 주소를 **저장**하는 변수 |

3. **포인터 연산자**

   | 연산자 | 이름            | 의미               |
   | :----: | --------------- | ------------------ |
   | **\*** | **간접 연산자** | 해당 주소의 **값** |
   | **&**  | **번지 연산자** | 변수의 **주소**    |
   - 연산자 사용

     ```c
     int a = 100;    // 변수 a에 100 저장
     int *b;         // 포인터 변수 b 선언
     b = &a;         // b에 a의 주소 저장
     printf("%d", *b);  // b가 가리키는 값 출력 → 100
     ```

4. **포인터 변수 선언**
   1. 형식

      ```c
      자료형 *변수명;
      ```

   2. 예시

      ```c
      int *p;      // 정수형 포인터 변수
      char *c;     // 문자형 포인터 변수
      float *f;    // 실수형 포인터 변수
      ```

5. **포인터 동작 원리 (시험 필수!)**
   1. 예시

      ```c
      int a = 100;
      int *b;
      b = &a;
      ```

   2. 메모리 구조

      ```
      메모리 주소:    4번지      8번지
      변수:          a           b
      값:           100          4
                    ↑            │
                    └────────────┘
                      b가 a의 주소(4)를 저장
      ```

   3. 각 표현의 의미

      | 표현    | 의미                            |   값    |
      | ------- | ------------------------------- | :-----: |
      | **a**   | 변수 a의 값                     | **100** |
      | **&a**  | 변수 a의 **주소**               |  **4**  |
      | **b**   | 포인터 변수 b의 값 (= a의 주소) |  **4**  |
      | **\*b** | b가 가리키는 곳의 **값**        | **100** |

6. **포인터와 배열 (시험 필수!)**
   1. 핵심 규칙
      - 배열 이름 = **첫 번째 요소의 주소**
   2. 예시

      ```c
      int a[5] = {10, 20, 30, 40, 50};
      int *b;

      b = a;       // a[0]의 주소를 b에 저장
      b = &a[0];   // 위와 동일!
      ```

   3. 배열과 포인터 관계

      | 표현        | 의미            |  값  |
      | ----------- | --------------- | :--: |
      | **a**       | a[0]의 주소     | 주소 |
      | **a[0]**    | 첫 번째 요소 값 |  10  |
      | **\*(a+0)** | a[0]과 동일     |  10  |
      | **\*(a+1)** | a[1]과 동일     |  20  |
      | **\*(a+2)** | a[2]과 동일     |  30  |

   4. 배열 요소 접근 공식

      ```
      a[i] = *(a + i)
      ```

7. **포인터 변수의 용도**

   | 용도                 | 설명                  |
   | -------------------- | --------------------- |
   | **연결 자료 구조**   | 링크드 리스트 등 구성 |
   | **동적 메모리**      | 힙 영역 접근          |
   | **배열 인수 전달**   | 함수에 배열 전달      |
   | **문자열 표현**      | 문자열 처리           |
   | **메모리 직접 접근** | 효율적 데이터 접근    |

8. **포인터 변수 특징**

   | 특징            | 설명                              |
   | --------------- | --------------------------------- |
   | **메모리 영역** | **힙(Heap)** 영역 접근            |
   | **변수 유형**   | **동적 변수**                     |
   | **크기**        | 주소 크기 (보통 4Byte 또는 8Byte) |

9. **포인터 예제**
   1. 예제 1: 기본 포인터

      ```c
      int a = 10;
      int *p = &a;

      printf("%d", a);    // 10
      printf("%d", &a);   // a의 주소
      printf("%d", p);    // a의 주소
      printf("%d", *p);   // 10
      ```

   2. 예제 2: 포인터로 값 변경

      ```c
      int a = 10;
      int *p = &a;

      *p = 20;            // p가 가리키는 곳(a)에 20 저장
      printf("%d", a);    // 20
      ```

   3. 예제 3: 배열과 포인터

      ```c
      int a[3] = {10, 20, 30};
      int *p = a;

      printf("%d", *p);       // 10 (a[0])
      printf("%d", *(p+1));   // 20 (a[1])
      printf("%d", *(p+2));   // 30 (a[2])
      ```

### 9. 구조체 (Structure)

1. **구조체란?**

   > **한 줄 정의:** **자료의 종류가 다른** 변수들을 하나로 묶은 것

2. ** 배열 vs 구조체**

   |  구분  |       배열        |           구조체            |
   | :----: | :---------------: | :-------------------------: |
   | 자료형 | **동일**한 자료형 |       **다른** 자료형       |
   |  예시  |     int a[5]      | 이름(문자), 급여(숫자) 혼합 |
   - 구조체 사용 예
     - 사원 정보: 이름(문자) + 직위(문자) + 급여(숫자) → **문자와 숫자 혼용** → 구조체!

3. **구조체 정의**
   - int, char 같은 **새로운 자료형**을 만드는 것
   - 형식

     ```c
     struct 구조체이름 {
         멤버1;
         멤버2;
         ...
     };
     ```

   - 예시

     ```c
     struct sawon {
         char name[10];   // 이름 (문자열)
         char jikwi[10];  // 직위 (문자열)
         int pay;         // 급여 (정수)
     };
     ```

     | 요소       | 설명                          |
     | ---------- | ----------------------------- |
     | **struct** | 구조체 정의 **예약어**        |
     | **sawon**  | 구조체 **이름** (사용자 지정) |
     | **멤버**   | 구조체 안의 **변수들**        |

4. **구조체 변수 선언**
   - 형식

     ```c
     struct 구조체이름 변수명;
     ```

   - 예시

     ```c
     struct sawon ansan;      // 일반 구조체 변수
     struct sawon *seoul;     // 구조체 포인터 변수
     ```

5. **구조체 멤버 지정**
   1. 일반 변수: **점(.)** 사용

      ```c
      ansan.name = "김한국";
      ansan.jikwi = "대리";
      ansan.pay = 4000000;
      ```

   2. 포인터 변수: **화살표(->)** 사용

      ```c
      seoul->name = "홍길동";
      seoul->jikwi = "과장";
      seoul->pay = 5000000;
      ```

   3. 포인터 변수: **(\*변수).멤버** 방식

      ```c
      (*seoul).name = "홍길동";
      (*seoul).jikwi = "과장";
      (*seoul).pay = 5000000;
      ```

6. **멤버 지정 연산자 비교**

   |    변수 종류    |     연산자      | 예시                |
   | :-------------: | :-------------: | ------------------- |
   |  **일반 변수**  |   **.** (점)    | ansan**.** name     |
   | **포인터 변수** | **->** (화살표) | seoul**->** name    |
   | **포인터 변수** |  **(\*)** 방식  | (\*seoul)**.** name |

7. **구조체 전체 예제**

   ```c
   #include <stdio.h>

   // 구조체 정의
   struct sawon {
       char name[10];
       char jikwi[10];
       int pay;
   };

   int main() {
       // 구조체 변수 선언
       struct sawon ansan;
       struct sawon *seoul;

       // 일반 변수로 멤버 지정 (.)
       ansan.name = "김한국";
       ansan.pay = 4000000;

       // 포인터 변수로 멤버 지정 (->)
       seoul = &ansan;
       seoul->pay = 5000000;

       return 0;
   }
   ```

8. **구조체 vs 배열 vs 포인터 비교**

   | 구분 |        배열         |   포인터   |         구조체          |
   | :--: | :-----------------: | :--------: | :---------------------: |
   | 목적 | 동일 자료형 여러 개 | 주소 저장  |  **다른 자료형** 묶음   |
   | 기호 |       **[ ]**       |   **\***   |       **struct**        |
   | 접근 |     a[0], a[1]      | _p, _(p+1) | **s.멤버**, **s->멤버** |

### 10. Python의 기초

1. **Python 기본 문법**
   - C언어와의 차이점 (시험 필수!)

     | 항목         |  C언어   |       Python       |
     | ------------ | :------: | :----------------: |
     | 자료형 선언  | **필수** |      **없음**      |
     | 세미콜론(;)  | **필수** |      **없음**      |
     | 코드 블록    | **{ }**  | **콜론(:) + 여백** |
     | 연속 값 저장 |   불가   |      **가능**      |

   - Python 특징
     - 변수 **자료형 선언 없음**
     - 문장 끝 **세미콜론(;) 불필요**
     - 코드 블록 = **콜론(:) + 여백(4칸 또는 탭)**
     - 같은 수준 코드 = **동일한 여백** 필수

     ```python
     # Python 코드 블록 예시
     if a > 0:
         print("양수")    # 4칸 여백
         print("출력")    # 동일 여백
     ```

2. **데이터 입출력 함수**
   1. input() 함수
      - 키보드로 입력받아 변수에 저장 (모든 값 **문자열**로 저장)

      ```python
      # 형식 1: 기본 입력
      변수 = input("출력문자")

      # 형식 2: 여러 개 입력 (분리)
      변수1, 변수2 = input().split()
      ```

   2. print() 함수
      - 값을 화면에 출력

      ```python
      # 형식 1: 기본 출력
      print(출력값1, 출력값2, sep="분리문자", end="종료문자")

      # 형식 2: 서식 문자열
      print("서식" % (값1, 값2))
      ```

3. **형변환 (Casting)**
   - input()은 **문자열**로 저장하므로 숫자 사용 시 **형변환** 필요
   - 데이터 1개 일때

     ```python
     변수 = int(input())      # 정수로 변환
     변수 = float(input())    # 실수로 변환
     ```

   - 데이터 2개 이상 (map 함수 사용)

     ```python
     변수1, 변수2 = map(int, input().split())     # 정수로 변환
     변수1, 변수2 = map(float, input().split())   # 실수로 변환
     ```

4. **리스트 (List)**
   - Python에서 배열 대신 사용하며, **다양한 자료형 혼합** 가능
   - 특징

     | 항목        | 배열 (C) | 리스트 (Python) |
     | ----------- | :------: | :-------------: |
     | 크기        | **고정** |    **가변**     |
     | 자료형      | **동일** |  **혼합 가능**  |
     | 시작 인덱스 |    0     |      **0**      |
     | 기호        |   [ ]    |     **[ ]**     |

   - 형식

     ```python
     리스트명 = [값1, 값2, 값3]
     리스트명 = list([값1, 값2, 값3])

     # 예시
     a = [1, 2.5, "hello"]   # 다양한 자료형 혼합 가능!
     ```

5. **딕셔너리 (Dictionary)**
   - **키:값** 쌍으로 연관된 값을 묶어 저장
   - 리스트 vs 딕셔너리

     | 구분 |      리스트       |    딕셔너리     |
     | :--: | :---------------: | :-------------: |
     | 기호 |      **[ ]**      |     **{ }**     |
     |  키  | 숫자 (0, 1, 2...) | **사용자 지정** |
     | 접근 |       a[0]        |     a["키"]     |

   - 형식

     ```python
     딕셔너리명 = {키1:값1, 키2:값2}
     딕셔너리명 = dict({키1:값1, 키2:값2})

     # 예시
     sawon = {"이름":"홍길동", "나이":25}
     print(sawon["이름"])   # 홍길동
     ```

6. **Range**
   - **연속된 숫자**를 생성 (반복문에서 많이 사용)
   - 형식

     ```python
     range(최종값)              # 0 ~ 최종값-1
     range(초기값, 최종값)       # 초기값 ~ 최종값-1
     range(초기값, 최종값, 증가값)  # 초기값부터 증가값만큼 증가

     # 예시
     range(5)        # 0, 1, 2, 3, 4
     range(1, 5)     # 1, 2, 3, 4
     range(1, 10, 2) # 1, 3, 5, 7, 9
     ```

7. **슬라이스 (Slice)**
   - 순차형 객체에서 **일부를 잘라** 반환
   - 형식

     ```python
     객체명[초기위치:최종위치]
     객체명[초기위치:최종위치:증가값]
     ```

   - 생략 가능

     ```python
     객체명[:]       # 전체
     객체명[초기위치:]    # 초기위치부터 끝까지
     객체명[:최종위치]    # 처음부터 최종위치-1까지
     객체명[::증가값]     # 증가값만큼 건너뛰며
     ```

   - 예시

     ```python
     a = [0, 1, 2, 3, 4, 5]

     a[1:4]    # [1, 2, 3]
     a[:3]     # [0, 1, 2]
     a[2:]     # [2, 3, 4, 5]
     a[::2]    # [0, 2, 4]
     ```

   💡 **주의:** 최종위치는 **포함하지 않음!**

8. **Python 자료구조 비교**

   | 구분 |  리스트  |   튜플   |    딕셔너리     |
   | :--: | :------: | :------: | :-------------: |
   | 기호 | **[ ]**  | **( )**  |     **{ }**     |
   | 변경 | **가능** | **불가** |    **가능**     |
   |  키  |   숫자   |   숫자   | **사용자 지정** |

### 11. Python의 활용

1. **Python 제어문/반복문 특징**

   > **한 줄 정의:** **콜론(:)**과 **여백**으로 코드 블록 구분 (중괄호 { } 없음!)

2. **if문**
   - 형식 1: 조건이 참일 때만 실행

     ```python
     if 조건:
         실행할 문장
     ```

   - 형식 2: 참/거짓 다른 문장 실행

     ```python
     if 조건:
         실행할 문장1
     else:
         실행할 문장2
     ```

   - 형식 3: 조건 여러 개 (elif 사용)

     ```python
     if 조건1:
         실행할 문장1
     elif 조건2:
         실행할 문장2
     elif 조건3:
         실행할 문장3
     else:
         실행할 문장4
     ```

   - 형식 4: 중첩 if문

     ```python
     if 조건1:
         if 조건2:
             실행할 문장1
         else:
             실행할 문장2
     else:
         실행할 문장3
     ```

   - C언어 vs Python if문

     |   구분    |    C언어    |     Python      |
     | :-------: | :---------: | :-------------: |
     | 다중 조건 | **else if** |    **elif**     |
     | 코드 블록 |   **{ }**   | **콜론 + 여백** |

3. **for문**
   - 형식 1: range 이용

     ```python
     for 변수 in range(최종값):
         실행할 문장
     ```

   - 형식 2: 리스트 이용

     ```python
     for 변수 in 리스트:
         실행할 문장
     ```

   - 예시

     ```python
     # range 이용
     for i in range(5):
         print(i)  # 0, 1, 2, 3, 4

     # 리스트 이용
     a = [10, 20, 30]
     for x in a:
         print(x)  # 10, 20, 30
     ```

   - C언어 vs Python for문

     |  구분  |         C언어         |            Python            |
     | :----: | :-------------------: | :--------------------------: |
     |  형식  | for(초기; 조건; 증가) | for 변수 **in** range/리스트 |
     | 키워드 |           -           |            **in**            |

4. **while문**
   - 형식

     ```python
     while 조건:
         실행할 문장
     ```

   - 예시

     ```python
     i = 0
     while i < 5:
         print(i)
         i += 1
     # 출력: 0, 1, 2, 3, 4
     ```

5. **클래스 (Class)**
   - 객체를 만들기 위한 **틀(설계도)**
   - 클래스 정의 형식

     ```python
     class 클래스명:
         실행할 문장

         def 메소드명(self, 인수):
             실행할 문장
             return 값
     ```

   - 객체 선언 형식

     ```python
     변수명 = 클래스명()
     ```

   - 예시

     ```python
     class Calculator:
         def add(self, a, b):
             return a + b

         def sub(self, a, b):
             return a - b

     # 객체 생성
     calc = Calculator()
     print(calc.add(3, 5))  # 8
     ```

   - 클래스 핵심 포인트

     | 요소       | 설명                               |
     | ---------- | ---------------------------------- |
     | **class**  | 클래스 정의 예약어                 |
     | **def**    | 메소드 정의 예약어                 |
     | **self**   | 자기 자신 참조 (필수 첫 번째 인수) |
     | **return** | 값 반환                            |

6. **클래스 없는 메소드 (함수)**
   - C언어의 **사용자 정의 함수**처럼 클래스 없이 단독 사용 가능
   - 형식

     ```python
     def 함수명(인수1, 인수2):
         실행할 문장
         return 값
     ```

   - 예시

     ```python
     def calc(x, y):
         x *= 3
         y /= 3
         print(x, y)
         return x

     a, b = 3, 12
     a = calc(a, b)
     print(a, b)

     # 출력:
     # 9 4.0  (함수 내부 print)
     # 9 12   (함수 외부 print)
     ```

   - 주의: 값에 의한 전달
     - Python에서 숫자, 문자열은 **값에 의한 전달**
     - 함수 내부에서 변경해도 원본 변경 안 됨! (b는 12 유지)

7. **C언어 vs Python 비교**

   |   구분    |      C언어      |      Python      |
   | :-------: | :-------------: | :--------------: |
   | 다중 조건 |     else if     |     **elif**     |
   |  반복문   |     for(;;)     |    for **in**    |
   | 함수 정의 | 반환형 함수명() | **def** 함수명() |
   | 코드 블록 |       { }       | **콜론 + 여백**  |
   | 자기 참조 |      this       |     **self**     |

8. **Python 제어문/반복문 요약**

   | 구문      | 형식              | 특징             |
   | --------- | ----------------- | ---------------- |
   | **if**    | if 조건:          | 조건 분기        |
   | **elif**  | elif 조건:        | 다중 조건        |
   | **for**   | for 변수 in 범위: | 반복 (in 사용)   |
   | **while** | while 조건:       | 조건 참인 동안   |
   | **def**   | def 함수명():     | 함수/메소드 정의 |
   | **class** | class 클래스명:   | 클래스 정의      |

### 13. 라이브러리

1. **라이브러리란?**

   > **한 줄 정의:** 자주 사용하는 **함수나 데이터**를 미리 만들어 모아 놓은 **집합체**

2. **라이브러리 특징**

   | 특징 | 설명                              |
   | ---- | --------------------------------- |
   | 목적 | **반복적인 코드 작성 방지**       |
   | 사용 | 필요할 때 **호출**하여 사용       |
   | 구성 | 도움말, 설치 파일, 샘플 코드 제공 |

3. **모듈 vs 패키지**

   |    구분    | 한 줄 정의                               |
   | :--------: | ---------------------------------------- |
   |  **모듈**  | 하나의 기능이 **한 개의 파일**로 구현    |

   | **표준 라이브러리** | 프로그래밍 언어에 **기본 포함**                    |
   | **외부 라이브러리** | 개발자들이 만들어 **인터넷에 공유**, 다운로드 필요 |

### 14. 예외처리

1. **예외 처리란?**

   > **한 줄 정의:** 프로그램 실행을 방해하는 **예외 발생 시** 미리 작성한 **처리 루틴을 수행**하도록 하는 것

2. **예외(Exception)란?**
   - 프로그램의 **정상 실행을 방해**하는 조건이나 상태
   - 예외 발생 원인

     | 원인                                        |
     | ------------------------------------------- |
     | 컴퓨터 **하드웨어** 문제                    |
     | **운영체제** 설정 실수                      |
     | **라이브러리** 손상                         |
     | **사용자 입력** 실수                        |
     | 받아들일 수 없는 **연산** (0으로 나누기 등) |
     | 할당하지 못하는 **기억장치 접근**           |

   - 일반적인 예외 처리
     - 프로그램 **종료**
     - **로그** 남기기

3. 예외 처리 지원 언어

   |    지원 방식    | 언어                           |
   | :-------------: | ------------------------------ |
   |  **내장 기능**  | C++, Ada, **JAVA**, JavaScript |
   | **조건문** 이용 | 그 외 언어                     |

4. **JAVA 예외 처리**
   - 특징
     - 예외를 **객체**로 취급
     - 예외 클래스는 **java.lang** 패키지에서 제공
     - **try ~ catch** 문 사용

5. **try ~ catch ~ finally 문**
   - 기본 형식

     ```java
     try {
         // 예외가 발생할 가능성이 있는 코드
     }
     catch (예외객체1 매개변수) {
         // 예외객체1 발생 시 처리 코드
     }
     catch (예외객체2 매개변수) {
         // 예외객체2 발생 시 처리 코드
     }
     catch (Exception 매개변수) {
         // 위 예외에 해당하지 않는 예외 처리
     }
     finally {
         // 예외 발생 여부와 관계없이 무조건 실행
     }
     ```

   - 구조 설명

     |    블록     | 역할                                  |
     | :---------: | ------------------------------------- |
     |   **try**   | 예외 발생 가능성 있는 코드            |
     |  **catch**  | 예외 발생 시 **처리 코드**            |
     | **finally** | **무조건 실행** (예외 발생 여부 무관) |

   - try ~ catch 특징

     | 특징         | 설명                                               |
     | ------------ | -------------------------------------------------- |
     | 예외 발생 시 | catch 블록으로 이동, **이후 코드 실행 안 함**      |
     | catch 변수   | 해당 **catch 블록에서만** 유효                     |
     | 중첩         | try ~ catch 안에 **또 다른 try ~ catch** 포함 가능 |
     | 중괄호       | **한 줄이라도 { } 생략 불가!**                     |

6.**JAVA 주요 예외 객체**

    | 예외 객체                          | 발생 원인                               |
    | ---------------------------------- | --------------------------------------- |
    | **ClassNotFoundException**         | **클래스**를 찾지 못한 경우             |
    | **NoSuchMethodException**          | **메소드**를 찾지 못한 경우             |
    | **FileNotFoundException**          | **파일**을 찾지 못한 경우               |
    | **InterruptedIOException**         | **입·출력** 처리가 중단된 경우          |
    | **ArithmeticException**            | **0으로 나누기** 등 산술 연산 예외      |
    | **IllegalArgumentException**       | **잘못된 인자** 전달                    |
    | **NumberFormatException**          | 숫자로 변환 불가한 **문자열→숫자** 변환 |
    | **ArrayIndexOutOfBoundsException** | **배열 범위** 벗어난 접근               |
    | **NegativeArraySizeException**     | 배열 크기를 **0보다 작게** 지정         |

7. **예외 객체 분류**
   1. NotFound 계열 (찾지 못함)

      | 예외 객체                  | 못 찾은 것 |
      | -------------------------- | :--------: |
      | **Class**NotFoundException | **클래스** |
      | **NoSuchMethod**Exception  | **메소드** |
      | **File**NotFoundException  |  **파일**  |

   2. 배열 관련

      | 예외 객체                          | 원인          |
      | ---------------------------------- | ------------- |
      | **ArrayIndexOutOfBounds**Exception | **범위 초과** |
      | **NegativeArraySize**Exception     | 크기 **음수** |

   3. 연산/변환 관련

      | 예외 객체                 | 원인               |
      | ------------------------- | ------------------ |
      | **Arithmetic**Exception   | **0으로 나누기**   |
      | **NumberFormat**Exception | 숫자 **변환 불가** |

   4. 기타

      | 예외 객체                    | 원인            |
      | ---------------------------- | --------------- |
      | **NullPointer**Exception     | **null** 참조   |
      | **IllegalArgument**Exception | **잘못된 인자** |
      | **InterruptedIO**Exception   | **입출력 중단** |

8. **예외 처리 예제**

   ```java
   public class Example {
       public static void main(String[] args) {
           try {
               int a = 10;
               int b = 0;
               int c = a / b;  // ArithmeticException 발생!
               System.out.println(c);  // 실행 안 됨
           }
           catch (ArithmeticException e) {
               System.out.println("0으로 나눌 수 없습니다");
           }
           finally {
               System.out.println("프로그램 종료");  // 무조건 실행
           }
       }
   }
   ```
