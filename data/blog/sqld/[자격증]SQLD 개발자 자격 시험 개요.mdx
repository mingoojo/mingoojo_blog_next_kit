2025년 마지막 SQLD 시험을 준비하며 정리한 공부 내용이다.

단순히 자격증 취득을 위한 공부가 아니라, **실무 적용**과 평소 추상적으로만 알던 개념을 체계적으로 정리하여  
효율적이고 안정적인 SQL 개발 역량을 갖추는 것을 목표로 한다.  

첫번째 개요에서는 SQL, SQLD에 정의와 시험을 위한 공부 범위 합격 기준등을 정리한다.

시험 관련 내용은 [한국데이터 산업 진흥원](https://www.dataq.or.kr/www/main.do)를 통해 확인할 수 있다.

---
### 1. SQL 정의

> **SQL (Structured Query Language)** : 데이터베이스(DB)에 직접 접근할수 있는 언어

SQL은 데이터를 정의(DDL), 조작(DML), 트랜잭션 제어(TCL), 권한 제어(DCL)하는 기능을 제공한다.  

    -   데이터 정의 (DDL)
    -   데이터 조작 (DML)
    -   트랜잭션 제어 (TCL)
    -   데이터 제어 (DCL)

---
### 2. SQLD 개발자의 직무

>**SQL Developer (SQLD)** : DB 및 데이터 모델링 지식을 바탕으로 응용SW 개발 시 정확하고 최적의 성능을 발휘하는 SQL 작성 가능자

1. **데이터모델의 이해 및 분석**
    -   데이터베이스 구성 핵심은 데이터모델
    -   건물 설계도와 같이 전체 DB 구조를 결정
    -   엔터티, 속성, 관계, 식별자, 정규화 등 이해 및 분석 작업 수행<br/><br/>

2. **SQL 이해 및 활용**
    -   SQL 문법, 옵티마이저, 인덱스 기본 원리 이해
    -   DDL을 통한 테이블 생성/변경/삭제
    -   DML을 통한 데이터 입력/조회/수정/삭제
    -   다양한 JOIN, 서브쿼리 활용


---
### 3. SQLD 자격 시험 과목 및 내용


1. **데이터모델의 이해**
    - 데이터모델 이해
    - 엔터티, 속성, 관계, 식별자
    - 정규화/반정규화와 성능
    - 대용량·분산 DB에 따른 성능 고려


2. **SQL 기본 및 활용**
    - **SQL 기본** : DDL, DML, TCL, WHERE, FUNCTION, GROUP BY, HAVING, ORDER BY, JOIN
    - **SQL 활용** : 표준조인, 집합연산자, 계층형 질의, 서브쿼리, 그룹 함수, 윈도우 함수, DCL, 절차형 SQL
    - **SQL 최적화 기본 원리** : 옵티마이저와 실행계획, 인덱스, 조인 수행 원리


---
### 4. 시험 배점 및 합격 기준


1. **시험 배점**
    - 데이터모델링 이해: 10문항, 20점
    -   SQL 기본 및 활용: 40문항, 80점
    -   총 50문항 / 90분 / 100점 만점


2. **합격 기준**
    -   총점 60점 이상
    -   과목별 40% 미만이면 불합격
