```bash
//목표 구조
LEVEL 1: 용어설명 (Terms)
│
├── LEVEL 2: SQLD 용어
│   ├── LEVEL 3: 데이터모델링
│   │   ├── 엔터티(Entity)
│   │   ├── 식별자(Identifier)
│   │   └── 정규화(Normalization)
│   ├── LEVEL 3: SQL 문법
│   │   ├── JOIN
│   │   ├── GROUP BY / HAVING
│   │   └── PIVOT / UNPIVOT
│   └── LEVEL 3: 성능 및 트랜잭션
│       ├── INDEX
│       ├── COMMIT / ROLLBACK
│       └── VIEW
│
├── LEVEL 2: 백엔드 용어
│   ├── LEVEL 3: Node.js
│   │   ├── 이벤트 루프(Event Loop)
│   │   ├── 논블로킹 I/O
│   │   └── 모듈 시스템
│   ├── LEVEL 3: 서버 아키텍처
│   │   ├── REST API
│   │   ├── Middleware
│   │   └── JWT / 인증
│
├── LEVEL 2: 프론트엔드 용어
│   ├── LEVEL 3: React
│   │   ├── 마운트(Mount)
│   │   ├── 렌더링(Rendering)
│   │   ├── 상태관리(State Management)
│   │   └── 훅(Hook)
│   ├── LEVEL 3: UI/UX
│   │   ├── 접근성(Accessibility)
│   │   ├── 반응형 디자인
│   │   └── Tailwind / Styled-components
│
├── LEVEL 2: 시스템 / 인프라 용어
│   ├── LEVEL 3: 운영체제
│   │   ├── 프로세스 / 스레드
│   │   ├── 스케줄러
│   │   └── 메모리 관리
│   ├── LEVEL 3: 서버 구조
│   │   ├── 클라이언트 / 서버
│   │   ├── 로드밸런서
│   │   └── 컨테이너(Docker)
│
├── LEVEL 2: 네트워크 용어
│   ├── LEVEL 3: 프로토콜
│   │   ├── TCP / UDP
│   │   ├── HTTP / HTTPS
│   │   └── DNS
│   ├── LEVEL 3: 네트워크 구조
│   │   ├── IP / Subnet
│   │   ├── 방화벽 / NAT
│   │   └── CDN
│
└── LEVEL 2: 기타 IT 용어
    ├── 클라우드(AWS / GCP / Azure)
    ├── 버전관리(Git, Branch, Rebase)
    ├── 데이터포맷(JSON, XML, YAML)
    └── 알고리즘 / 자료구조
```
